# Pomodoro

<https://physics-htfi.github.io/Pomodoro/>

ポモドーロテクニック用のタイマー。

- 時間は 25 分 & 5 分で固定。
- 開始時や時間がゼロになった時に効果音が鳴る。
- 操作：
  - 画面のクリックで「開始」「一時停止」。
  - カウント値テキストクリックでタイマーリセット。
  - ←→ キー：残り時間 ±1 分。
  - ↑↓ キー：カウント ±1。
  - ⚙️ 「カウント値の保存先」と「効果音の出力先」を変更。
  - 🪟 ピクチャインピクチャ表示。
